<h1 align="center">zzz 😴</h1>

<p align="center">
A deliberately lazy programming language that does just enough.
</p>

zzz is a lightweight browser-based programming language featuring a custom interpreter, interactive terminal, and guided learning experience.

## Features

- Simple syntax
- Variables without declarations
- Arrays
- Conditional logic
- While loops
- Functions
- Boolean values (`ya`, `nah`)
- Null value (`idk`)
- Interactive terminal
- Learn page with examples
- Runs entirely in the browser

---

## Language Examples

### Hello World

```zzz
zzz

yawn "hello world"

nap
```

### Variables

```zzz
zzz

name = "world"
age = 25

yawn "hello " + name
yawn age

nap
```

### Conditionals

```zzz
zzz

energy = 3

if u care energy > 5
  yawn "let's go"
dont bother
  yawn "maybe later"
whatever

nap
```

### Loops

```zzz
zzz

i = 0

while still awake i < 5
  yawn i
  i = i + 1
im done

nap
```

### Functions

```zzz
zzz

whenever greet
  yawn "hello"
thats it

do greet

nap
```

## Keywords

| Keyword | Purpose |
|----------|----------|
| zzz | Start program |
| nap | End program |
| yawn | Print output |
| ya | True |
| nah | False |
| idk | Null |
| if u care | If statement |
| dont bother | Else statement |
| whatever | End if |
| while still awake | While loop |
| im done | End loop |
| im out | Break |
| whenever | Define function |
| thats it | End function |
| do | Call function |

## Running Locally

Clone the repository:

```bash
git clone https://github.com/ZannatulRaian/zzz-lang.git
```

Open:

```bash
index.html
```

or run a local server:

```bash
npx serve
```

---

Build settings:

```text
Build command:
(none)

Publish directory:
.
```

Since zzz is a static website, no build process is required.
