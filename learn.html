01

Variables

No let, no const, no var. Just name something and give it a value. zzz doesn’t care what type it is.

variables.meh
zzz // just name it. that's it. name = "world" age = 25 tired = ya nothing = idk yawn "hello " + name yawn "age: " + age yawn tired yawn nothing nap
You can reassign variables anytime. zzz won’t stop you. It’s not your parent.
02

If Statements

Sometimes you need to make a decision. Use if u care to check something. Else? dont bother. End it all with whatever.

decision.meh
zzz energy = 3 if u care energy > 5 yawn "let's do this" dont bother yawn "maybe later" yawn "...or never" whatever nap
You can nest if u care blocks inside each other. zzz handles the nesting. You handle the existential dread.
03

Arrays

Need to store multiple things? Wrap them in square brackets. Access with index starting from zero. That part isn’t lazy. Sorry.

snacks.meh
zzz snacks = ["chips", "cookies", "soda"] yawn snacks[0] yawn snacks[1] yawn snacks[2] nap
Arrays can hold mixed types: numbers, strings, booleans, even other arrays. zzz is too tired to judge.
04

Loops

When doing something once isn’t enough, use while still awake. It keeps going until the condition is false. End with im done.

loop.meh
zzz i = 0 while still awake i < 5 yawn "counting... " + i i = i + 1 im done yawn "finally. nap time." nap
Use im out inside a loop to break early. Like leaving a meeting that should’ve been an email.
05

Functions

Want to reuse code? Define with whenever name, end with thats it, call with do name. Functions share all variables globally — no parameters, no return values. Just vibes.

func.meh
zzz whenever greet yawn "hey there" yawn "welcome to zzz" thats it do greet do greet nap
Functions must be defined before they’re called. Even zzz has limits.
